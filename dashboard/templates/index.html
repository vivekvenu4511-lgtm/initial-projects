🤗 HuggingFace Model Manager
Aria v7 uses llama-cpp-python to run models directly — no Ollama required. Download GGUF models from HuggingFace, and Aria will automatically allocate GPU layers based on your hardware (VRAM, RAM, CPU threads).

Quick start:
1. Run python setup_v7.py to install the GPU-optimised backend
2. Click 🤗 Open Model Manager to download a model
3. Start chatting instantly — no CLI needed
